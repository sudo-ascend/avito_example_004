# Bloom Atelier

Статический одностраничный сайт для цветочного магазина: hero, каталог букетов, блок доверия, преимущества, отзывы, FAQ и форма заявки.

## Запуск

Сборка не нужна.

1. Откройте `index.html` в браузере.
2. Если удобнее, запустите сайт через любой локальный static server или VS Code Live Server.

## Структура

- `index.html` — разметка и основной контент сайта
- `styles.css` — стили, адаптив, визуал и анимации
- `script.js` — интерактивность, валидация формы, контакты, меню, модальное окно
- `assets/images/` — фотографии букетов и мастерской
- `assets/favicon.svg` — favicon

## Где менять контент

### Контакты

В начале `script.js` есть объект `siteConfig`. Там меняются:

- название магазина
- телефон
- WhatsApp
- Telegram
- адрес
- часы работы

### Тексты, цены и карточки

Меняются прямо в `index.html`.

Главные зоны:

- первый экран
- каталог
- блок "О нас"
- преимущества
- отзывы
- FAQ
- форма и footer

### Фото

Все изображения лежат в `assets/images/`.

Чтобы заменить фото:

1. добавьте новый файл в `assets/images/`
2. обновите путь в `index.html`
3. при необходимости поменяйте `alt`

## Форма

Сейчас реализованы UI и валидация без backend.

В `script.js` уже оставлена точка подключения backend внутри `setupForm()`:

- данные собираются в объект `payload`
- временная отправка сделана через `setTimeout`
- позже это можно заменить на `fetch(...)`

## SEO

Уже добавлены:

- `title`
- `description`
- `og:title`
- `og:description`
- `og:image`
- `alt` у изображений
- семантические теги `header`, `main`, `section`, `footer`

## Примечание

Шрифты подключаются через Google Fonts, поэтому для полного совпадения дизайна нужен доступ в интернет.
